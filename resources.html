Git and GitHub

git is a very simple programming language for version control, run from the command line. GitHub is a popular website where you can use git to collaborate with others on code-based projects. It's basically git + the internet. Your code is hosted in online repositories ("repos") which can be public (free) or private (not free). You normally don't store data, Word documents, or other stuff on GitHub. It works best with plain text files.

d3.js

Ever admired a beautiful infographic on The New York Times or FiveThirtyEight? It was probably built in d3.js. d3.js is a JavaScript library that binds data onto HTML elements. It takes almost any sort of data, and can make almost any sort of graph - you just need some JavaScript coding skills and imagination. It helps to be familiar with HTML/CSS and JavaScript before getting started; but some d3.js tutorials assume no prior knowledge.

F(AF)L
None yet. Interested in teaching?
Example code
d3.selectAll("circle")
  .data(data)
  .enter()
  .append("circle")
    .attr("cx", function(d) { return d.year; })
    .attr("cy", function(d) { return d.income; })
    .attr("r", function(d) { return d.population; })
    .attr("color", "red");

Generating interactive, online data visualizations of nearly any type (maps, charts, etc).

SQL

When dealing with relational databases, you use SQL (Structured Query Language, pronounced "sequel") to manage the data. In general, it's probably a good idea to get experience with different types of data: for example, XML and JSON data.
